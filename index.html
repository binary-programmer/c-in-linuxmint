How to Run a C Program in Linux-Mint

Let's follow the following steps to learn how to run a C program in Linux-mint:

Step-1: Open Terminal

To open terminal press Ctrl+Alt+t together. Here, the plus-sign ('+') indicates together i.e. we have to press Ctrl, Alt and t keys together from keyboard but not the plus key. Terminal will open like the following figure:

Figure-1: Screenshot of Mint Terminal

Step-2: Checking gcc Version

Type the following command in the terminal as shown in Figure-2 to check whether gcc compiler is installed or not before installing it:

 gcc --version

Figure-2: Checking gcc Version

Then press Enter key from keyboard.

Step-3: If gcc Not Found

If gcc not found comes like the following message as shown in Figure-3, follow from Step-4. Otherwise, follow from Step-x.

 Command 'gcc' not found, but can be installed with:
 sudo apt install gcc

Figure-3: Checking gcc Version

Step-4: Run Command: sudo apt install gcc

Type the following command in the terminal as shown in Figure-4 to install gcc.

 sudo apt install gcc

Figure-4: Run Command: sudo apt install gcc

Then press Enter key from keyboard.

Step-5: Type Password

The terminal will ask to type password like the following message as shown in Figure-5.

 [sudo] password for binary-programmer

Figure-5: Type Password

Now, type the user password and press Enter key from keyboard. Here, my user name is binary-programmer. So, I am going to type the password of binary-programmer.

Step-6: gcc Installation Completion

The terminal will show the message as shown in Figure-6 indicating the gcc installation completion.

Figure-6: gcc Installation Completion

Step-7: Recheck gcc Version

Recheck gcc version like Step-2: as shown in Figure-7.

 gcc --version

Figure-7: Recheck gcc Version

Step-8: gcc Version Shown

If gcc version is shown as shown in Figure-8, it can be said that gcc installation has been successful.

Figure-8: gcc Version Shown
